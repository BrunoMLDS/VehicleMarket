# VehicleMarket

Criadores:

João Santos

Bruno Santos

Gonçalo Oliveira

VehicleMarket é uma app de compra e venda de carros que liga compradores e vendedores de forma simples e segura. Permite publicar anúncios, pesquisar com filtros avançados e contactar diretamente. Interface intuitiva e rápida para encontrar o carro ideal.
